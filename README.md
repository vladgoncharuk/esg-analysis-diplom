ESG & Financial Analysis (Russia vs S&P 500)

📊 Project Description
This project analyzes the relationship between ESG metrics and financial performance of companies from Russia and the S&P 500 using machine learning methods.

🎯 Objective
- Compare ESG impact on financial indicators
- Identify key drivers of company performance
- Analyze differences between Russian and US markets

🧠 Methods
- Data preprocessing (handling missing values, normalization)
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Machine Learning models (Regression / etc.)

📈 Results
- ESG factors show different levels of impact in Russia vs US
- Financial indicators are more strongly correlated in US companies
- Identified key features influencing company valuation

🛠️ Tools
- Python (pandas, numpy, sklearn)
- matplotlib / seaborn
- Google Colab

📁 Files
- `Diplom.ipynb` — main analysis notebook
